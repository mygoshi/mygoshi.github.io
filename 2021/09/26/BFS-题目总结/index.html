BFS 题目总结

算法介绍

BFS（广度优先算法）是指一层层地进行遍历，先将一层中所有节点进行遍历，再以本层遍历结果为起点对下一层进行遍历，已经遍历过的节点不会再次被遍历。

实现 BFS 时需要用到以下两个变量：

  • 队列：先进先出，用于存储待遍历的节点；
  • 标记：一般是 boolean 类型的数组，用于存储某个节点是否被遍历过。

使用 BFS 可以求解最短路径等最优解的问题：第一次遍历到目的节点，其所经过的路径为最短路径。

Leetcode.127 单词接龙

若两单词仅相差一个字母，则可认为两单词间有一条边相连，根据这一想法就可以构建出一张图，这样题目中要求的“找到从 beginWord 到 endWord 的 最短转换序列 中的 单词数目”就可以转化为“在图中找到从 beginWord 到 endWord 的最短路径”，再使用 BFS 算法解决。

Leetcode.279 完全平方数

这道题当时卡的有点久，起初是因为将图的概念忘记了，所以完全没有任何思路。在做完上边的一道题后，我想起了用图的思想来解决这道题，但因为思路的偏差，还是没有任何进展，无奈只能看答案勉强将这道题做出来🤦‍♂️。

如果使用图 + BFS 算法来解决这道问题，应该将从 0 到 n 的数字视为图中的节点，其中下级节点与上级节点相差一个完全平方数，第一次遍历到 0 所经过的级数就是题目所求。在做这道题时不能将节点设置为完全平方数来求其和判断是否等于 n，因为这样每个完全平方数只会被遍历到一次，而实际情况是有的完全平方数出现了多次。

这道题还能用动态规划的方法来解决，后面还会用这种方法来解决。
