Leetcode.19 - 删除链表的倒数第 N 个结点

本文最后更新于：3 分钟前

题目描述

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

示例：

原文出处：力扣

思路分析

遍历

思路

正向遍历链表得到链表长度 size, 然后迭代 size-n+1 次，删除该节点的下一个节点。

双指针

思路

使用 first 和 second 两个指针，使 second 指针比 first 指针提前 个，当 second 指针指向最后一个时，删除 first 指针指向的节点。先让 second 指针开始遍历，遍历 次后，再让两个指针同时遍历，可以使 second 指针比 first 指针提前 个。
